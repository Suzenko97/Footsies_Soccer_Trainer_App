To run the app:
    From root directory, in cmd line type "npm run dev"

Technologies used so far:
    React
    Cors
    Express
    Concurrently (not a full tool but useful to enable easy start)
    Node
    Firebase (For db and login)
    React Roter Dom


    ***DB ACCESS RULES UPDATED AS OF 03/02/25***
